# Claude Prompts & Skills

A collection of reusable prompts and skills for Claude AI.

## Skills

| Skill | Description | Install |
|-------|-------------|---------|
| [memory-init](memory-init/) | Persistent memory system for Claude Code | `npx skills add spudy-vibing/Claude-Code-Prompts/memory-init` |

### Installing Skills

```bash
# Install a skill
npx skills add spudy-vibing/Claude-Code-Prompts/<skill-name>

# Use in Claude Code
/<skill-name>
```
